# HarmonyOS 激励数据看板插件使用说明

这是一个用于 2026 鸿蒙开发者激励计划达标数据分析的 Chrome 插件。安装后，插件会在华为开发者激励计划数据页面中辅助读取应用数据，并在页面表格中展示「本月月活」、本月增量，以及暂不满足应用的本月评分数据。

适用页面：

```text
https://developer.huawei.com/consumer/cn/activity/harmonyos-incentive/data-query2026
```

## 安装插件

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`。
2. 打开右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择本插件所在文件夹。
5. 安装完成后，浏览器工具栏中会出现插件图标。

## 使用步骤

1. 打开华为开发者激励计划数据页面。
2. 按页面提示登录华为开发者账号，并完成必要的安全校验。
3. 等待页面中的「开发者激励计划数据信息」表格加载完成。
4. 插件会自动在「上架应用数据」表格中追加「本月月活」列。
5. 状态为「暂不满足」的应用，会自动在「有效月活及评分」列显示为 `5.0 / 1 评` 这种形式。
6. 点击浏览器工具栏中的插件图标，可以查看最近捕获的数据概览。

## 弹窗按钮说明

- 「获取全部数据」：主动读取当前账号下的激励计划应用数据。
- 「插入本月月活列」：在页面表格中重新插入或刷新「本月月活」列。
- 「调试读取当前页明细」：调试使用，会逐行打开页面明细读取当前页数据；正常使用无需点击。
- 「重新扫描页面」：重新读取当前页面表格内容。

## 页面筛选

插件会在「有效月活及评分」表头中加入筛选下拉框，支持「全部 / 已达标 / 未达标」。选择「已达标」或「未达标」时，插件会先读取全量应用数据，再用筛选结果临时替换当前表格行，并显示命中数量；选择「全部」会恢复页面原本分页视图。

## 数据显示规则

- 「本月月活」显示应用在当前月份最新一次返回的月活数据。
- 插件会把每个应用本月第一次读取到的月活作为本月基准。
- 如果后续读取到的月活大于本月基准，会在数字右侧显示绿色增量，例如 `90 +20`。
- 进入新的月份后，插件会重新建立当月基准。
- 「有效月活及评分」列中，官方状态为「满足」的应用保持显示「满足」。
- 官方状态为「暂不满足」的应用，会用最新月份的「月末评分 / 月末评分个数」替换为 `5.0 / 1 评`；评分为空时显示 `暂无 / 0 评`。
- 如果暂时没有读取到接口数据，新列会默认显示 `0`。

## 常见问题

### 插件弹窗提示暂无数据怎么办？

请先确认已经登录华为开发者账号，并且目标页面表格已经加载完成。可以刷新页面后等待几秒，再点击插件弹窗中的「获取全部数据」或「重新扫描页面」。

### 为什么只看到当前分页的数据？

如果页面本身只加载了当前分页，可以点击插件弹窗中的「获取全部数据」，让插件主动读取完整应用数据。

### 插件会保存账号密码吗？

不会。插件只在已登录的华为开发者页面中读取页面数据和接口返回结果，不读取、不保存账号密码。

## 注意事项

- 插件依赖华为开发者页面当前的数据结构。如果页面后续改版，部分功能可能需要更新插件后才能继续使用。
- 如果账号没有权限、登录失效，或页面本身返回空数据，插件也无法展示对应数据。

## 项目结构

```text
manifest.json
src/
  content/    页面内容脚本，负责读取数据、操作表格、渲染筛选和月活列
  injected/   注入到页面主环境的脚本，负责捕获华为页面接口返回
  popup/      插件弹窗页面、样式和交互脚本
```
